Fourier Transforms

Introduction

The fourier transform is a model with a large variety of applications including quantum mechanics, signal analysis and the evaluation of differential equations.

Structure

A basic fourier transform is continuos function that can be evaluated for any real number $\zeta$. The shape of the function is determined by a series of magnitudes, denoted as $m$, frequencies, denoted as $\lambda$, and inital angles, denoted as $\theta$.

Function of a fourier transform

The function of a fourier transform is a sum of sines and cosines. The visualization uses circles to represent a combination of a magnitude, inital theta and frequency. The function itself can take the form of:
$$\hat{f}(\zeta) = \sum_{i=1}^{n}m_i * sin(t\lambda_i + \theta_i)$$
$t \epsilon [0, 2\pi]$.

When $t = 0.5$, $\vec{a}$ will be halfway between $\vec{c_n}$ and $\vec{c_{n - 1}}$, and when $t = 0$, $\vec{a}$ will equal $\vec{c_{n - 1}}$. So to create smooth intepolation, $t$ can be increased by a small amount every frame during the animation.

Interpolation between three points

To interpolate between between any two, two dimensional points a parametric equation can be used. To intepolate between any three, two dimensional points, multiple parametric equations can be used. If the result of this interpolation were animated, the path $\vec{a}$ follows, is a curve. To find the position of a at any $t$, it is useful to define the parametric equation as the function:

\(f_{\vec{c_1}, \vec{c_2}}(t) = \vec{c_1} + t(\vec{c_1} - \vec{c_2})\)
$t \epsilon [0, 1]$.


With the control points $\vec{c_1}, \vec{c_2}, and \vec{c_3}$, $\vec{a}$ is equal to:

$\vec{p_1} = f_{\vec{c_1}, \vec{c_2}}(t)$
$\vec{p_2} = f_{\vec{c_2}, \vec{c_3}}(t)$
$\vec{a} = f_{\vec{p_1}, \vec{p_2}}(t)$.
